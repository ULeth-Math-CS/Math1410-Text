{ $z = 2\sqrt{2} - 2i\sqrt{2}$}
{ $z = 2\sqrt{2} - 2i\sqrt{2} = 4\operatorname{cis}\left(\frac{7\pi}{4}\right)$, \, $\text{Re}(z) = 2\sqrt{2}$, \, $\text{Im}(z) = -2\sqrt{2}$, \, $|z| = 4$, \, $\text{arg}(z) = \left\{\frac{7\pi}{4} + 2\pi k \, | \, \text{$k$ is an integer} \right\}$ and $\text{Arg}(z) = -\frac{\pi}{4}$.}

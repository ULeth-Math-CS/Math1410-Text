{ $z = -5-2i$}
{ $z = -5-2i = \sqrt{29}\operatorname{cis}\left(\pi + \arctan\left(\frac{2}{5}\right)\right)$, \, $\text{Re}(z) = -5$, \, $\text{Im}(z) = -2$, \, $|z| = \sqrt{29}$, \, $\text{arg}(z) = \left\{\pi +\arctan\left(\frac{2}{5}\right) + 2\pi k \, | \, \text{$k$ is an integer} \right\}$ and $\text{Arg}(z) =  \arctan\left(\frac{2}{5}\right) -\pi $.}

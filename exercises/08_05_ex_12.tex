{ $z = i\sqrt[3]{7}$}
{ $z = i \sqrt[3]{7} = \sqrt[3]{7}\operatorname{cis}\left(\frac{\pi}{2}\right)$, \, $\text{Re}(z) =0$, \, $\text{Im}(z) = \sqrt[3]{7}$, \, $|z| = \sqrt[3]{7}$, \, $\text{arg}(z) = \left\{\frac{\pi}{2} + 2\pi k \, | \, \text{$k$ is an integer} \right\}$ and $\text{Arg}(z) = \frac{\pi}{2}$.}
